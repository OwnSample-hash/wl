# Store

## Használat

```sh
docker compose up
```

Alapértelmezetten a `8080` porton elérhető a [docker-compose.yaml](/docker-compose.yml) szerkeszthető

A configurácós fájl a `config/` található

## Olalak

- Index /
- Termék vásárlás /buy/{id}
- Licence /license
- Felhasználók /admin/users
- Admin felűlet /admin
